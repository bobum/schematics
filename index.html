Welcome to the Dark Mode Demo

This application demonstrates a fully functional dark mode toggle with smooth transitions and persistent settings.

Click the settings button in the header to toggle between light and dark modes.

Features

  • Toggle persists across browser sessions using localStorage
  • Smooth transition animations between modes
  • All components fully support both light and dark themes
  • Accessible and user-friendly interface
